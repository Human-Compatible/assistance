Updates the state of a given cell on the board based on the number of alive neighbors.

Parameters
----------
board : list[list[int]]
    The current generation of the board as a list of lists of integers.
row : int
    The row index of the cell to be updated.
col : int
    The column index of the cell to be updated.
num_alive_neighbors : int
    The number of alive neighbors for the given cell.

Returns
-------
int
    The new state of the cell (0 for dead, 1 for alive).
